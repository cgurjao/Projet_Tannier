File list
-----------------
Alignment.txt		Alignment of medieval and modern genomes
Blocks.py		Generates input file for GenDist and shows dotplots if asked
GenDist.py		Automatically runs the 'Blocks.py' script and returns genomic distance
 

Userguide
-----------------
To create the input file and find the genomic distance between the aligned genomes (minimum number of inversion), 
just run the GenDist.py script from the terminal

> python GenDist.py
